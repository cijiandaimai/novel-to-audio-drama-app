# AI 影视级音频提示词写作指导摘要

影视级音频提示词的核心不是“把对白写出来”，而是同时调度：

- 人声表演
- 音乐
- 环境声
- 拟音音效
- 空间感
- 情绪递进
- 时间节奏
- 混音层级

推荐结构：

```text
【音频类型】
【时长】
【整体情绪】
【场景空间】
【音乐设计】
【环境声】
【角色声线】
【对白内容】
【表演要求】
【音效时间线】
【混音要求】
【禁止事项】
```

关键原则：

1. 对白始终为前景主声部。
2. 音乐和环境声保持在背景层，不遮挡关键词。
3. 复杂动作戏必须使用时间轴。
4. 角色声线使用原创合成声线，不模仿任何真实演员、主播或公众人物。
5. 单条提示词尽量控制在 800-1800 字符。
6. 多人戏、动作戏和情绪转折密集的场景应拆成 30-90 秒片段。
