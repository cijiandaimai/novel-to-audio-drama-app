    傻瓜式配置

    把自己的 API 接进来

    没有 Key 时可先用演示模式；准备生成真实音频时再切到 HTTP。

    创作方案

    自动推荐会根据已配置模型选择 A/B/C 计划；大陆用户可优先配置豆包、千问和 Kimi。

    自动推荐规则：A 可用走 A；否则千问或 Kimi 可用走 C；最后用豆包 B 计划兜底。

    Doubao-音频生成-1.0

    最核心配置：逐段生成对白、音乐、音效和环境声一体化音频。

    官方音频生成 HTTP 使用 X-Api-Key 鉴权；模型 ID 当前填 seed-audio-1.0。

    Doubao-语音识别

    用于语音输入，把麦克风录音转成小说内容、提示词或酒馆对话。

    优先使用新版 X-Api-Key；旧版控制台才需要 App Key + Access Key。正式发布建议放在后端中转里。

    通用语音网关

    接入 epidemicsituation 语音 API，可记录 TTS、声音克隆和 ASR 的统一授权配置。

    该网站文档写明 Base URL 为 https://epidemicsituation.pages.dev，鉴权支持 Authorization: Bearer 或 x-api-key，会话接口为 POST /api/v1/auth/session。

    GPT：总控与剧本

    用于故事骨架、剧本 1、剧本 2 和音频提示词拆分。

    购买后复制 API Key，粘贴到这里。接口地址可用默认，也可填兼容接口。

    第三方 GPT 兼容服务

    没有 OpenAI 官方 API 时，可以接入 Cockpit Tools、One API、New API 等 OpenAI 兼容入口。

    电脑本地测试可填 localhost；安卓真机里的 localhost 是手机自身，若第三方服务跑在电脑上，请改成同一局域网内电脑 IP，例如 http://192.168.x.x:50360/v1。

    DeepSeek：酒馆与客服闲聊

    官方 DeepSeek Key，主要用于酒馆 API 模式、白泽小助手闲聊和低成本中文对话。

    DeepSeek 使用 OpenAI 兼容格式。也可填 https://api.deepseek.com，App 会自动补 /chat/completions。

    Gemini：场景扩写

    用于补足剧情、场景动作和戏剧冲突。

    在 Google AI Studio 获取 Key；留空接口时会自动拼接官方 generateContent 地址。

    豆包文本：中文台词

    用于中文对白润色、角色语气统一。

    在火山方舟开通模型服务并创建 API Key，然后填入模型 ID。

    千问 Qwen：结构化剧本

    C 计划增强项：适合角色表、场次表、剧本整理和 JSON 音频提示词。

    在阿里云百炼开通 Qwen；如果只想用国产增强方案，填千问、Kimi、豆包即可。

    千问 TTS：酒馆配音

    用于酒馆角色回复的一键朗读，返回音频后直接在 App 内播放。

    千问 TTS 非流式接口会返回音频 URL，有效期通常为 24 小时。北京地域需要替换真实 WorkspaceId。

    Kimi：长文理解

    C 计划增强项：适合长篇小说、人物关系、伏笔和章节脉络梳理。

    进入 Kimi API 平台创建 Key。C 计划会优先用它做长文理解和分场扩写。

    后端中转与网络

    正式使用建议把 Key 放在自己的后端。App 会尊重系统代理和你填写的中转接口。

    填了中转地址后，创作、直接生成音频、历史和联网诊断都会优先请求后端的 /api 接口；中转失败时不会偷偷改走本机直连，避免付费 Key 暴露在 APK 里。

    AI 客服与图片生成

    悬浮小球优先调用 GPT；GPT 不可用时回退豆包。图片功能建议正式版走后端中转。

    如果要“修改图片”，先在悬浮客服里上传参考图，再输入修改要求；没有图片模型时会返回可复制的图片提示词。

    白泽小助手 使用说明 / AI 客服

    新手入门

    1. 1到 API 配置页填写 Key、模型 ID 和接口地址。
    2. 2点输入框旁边的“测”，先检查网络能不能连上。
    3. 3API 输入后会自动保存；需要确认时可以点“测”做连通检查。
    4. 4回到创作页上传小说，或到酒馆切换 API 模式开始对话。

    常用提醒

    • A悬浮球可以拖到顺手的位置；长按 10 秒会隐藏，在首页切换语言会再次出现。
    • B大陆线路优先填豆包、DeepSeek、千问、Kimi；DeepSeek 适合酒馆和小助手闲聊。
    • C安卓真机访问电脑本地服务时，不要填 localhost，要填电脑局域网 IP。
    • D正式发布给别人使用时，尽量把付费 Key 放到自己的后端，不要直接打进 APK。
    如何开始创作

    先到 API 配置填写需要的模型 Key；只用中国大陆线路时，优先填豆包文本、豆包音频、千问和 Kimi。回到创作页上传小说，点击自动执行完整方案。

    API 配置与测试

    每个 API 输入框旁有“?”和“测”。“?”看官网和填写方法；“测”只检查网络连通，真正能不能生成还要看 Key、余额、模型权限和接口格式。填写后会自动保存在本机。

    酒馆怎么用

    酒馆可以本地运行，也可以切换 API。角色回复旁的“配音”会调用千问 TTS，把角色台词直接变成可播放音频。

    音箱播放器

    音箱页可以导入本地音频、歌词和背景图，也可以扫描下载文件夹。播放器支持播放列表、循环、变速、歌词滚动和全屏歌词。

    音频剪辑

    剪辑页支持双轨导入、拖动音块、调整播放头、录音、撤销和混音导出。移动端优先用顶部快捷按钮导入，再在轨道里长按拖动。

    直接生成音频

    如果已经有分段提示词，可以在创作页选择跳过编剧流程，直接上传提示词生成音频；生成后会按顺序保存，并可放进播放器收听。

    图片功能

    在“图片”页输入要求，可生成插画、封面、播放器背景，也可以上传参考图后让模型改图。优先 GPT，失败后尝试豆包图片模型。

    使用说明

    白泽声工坊是框架型工具

    要使用完整 AI 功能，需要去对应模型官方购买额度，并在 API 配置页填入自己的 Key。

    本应用不上传、不云端储存个人信息；个人配置、播放列表、导入记录等只保留在本机。

    调用第三方模型时，你提交的文本会按你配置发送到对应官方接口或后端中转服务，请自行确认服务条款。
